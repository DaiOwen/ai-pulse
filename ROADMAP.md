## 🔮 待优化项（基于产品经理评审）

### Phase 8: 内容层增强 ⏳

| 问题 | 建议 | 优先级 |
|------|------|:------:|
| 信源可访问性不稳定 | 增加更多备用信源（GitHub API、RSS Hub） | P0 |
| 搜索索引需动态更新 | 生成时自动更新 search-index.json | P1 |
| 内容时效性标注 | 每条新闻显示「X小时前」相对时间 | P2 |

### Phase 9: 分发层增强 ⏳

| 问题 | 建议 | 优先级 |
|------|------|:------:|
| Email/IM 推送未实现 | 增加 Email/微信/Telegram 推送 | P1 |
| PWA Push Notification | 增加浏览器推送通知 | P2 |
| 微信分享卡片优化 | 生成微信专属分享图片 | P2 |

### Phase 10: 数据层增强 ⏳

| 问题 | 建议 | 优先级 |
|------|------|:------:|
| 无统计追踪 | 增加隐私友好的访问统计 | P2 |
| 归档体积增长 | 超过30天的归档自动压缩 | P2 |
| 生成成功率未追踪 | 增加健康检查和告警机制 | P1 |

### Phase 11: 用户体验细节 ⏳

| 问题 | 建议 | 优先级 |
|------|------|:------:|
| 日历入口不明显 | 增加「📅 历史」更显眼入口 | P1 |
| 移动端手势支持 | 下拉刷新、左滑返回 | P2 |
| 打印优化样式 | print.css 优化打印输出 | P2 |
| 无障碍访问 | WCAG 合规性检查 | P2 |
