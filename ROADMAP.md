## 🎯 新增功能一览

### 用户交互
| 功能 | 触发方式 | 效果 |
|------|----------|------|
| 返回顶部 | 点击右下角 ↑ | 平滑滚动到页面顶部 |
| 收藏新闻 | 点击 ☆ 按钮 | 保存到 favorites.html |
| 分享新闻 | 点击 🔗 按钮 | 复制锚点链接 |
| 搜索历史 | 点击 🔍 搜索 | search.html 全文搜索 |
| 偏好设置 | 点击 🏷️ 设置 | settings.html 标签选择 |
| 下拉刷新 | 移动端下拉 | 重新加载页面 |

### 快捷键
| 按键 | 功能 |
|------|------|
| `J` / `↓` | 下一条新闻 |
| `K` / `↑` | 上一条新闻 |
| `T` | 切换暗/亮主题 |
| `?` | 显示快捷键帮助 |
| `/` | 聚焦搜索框 |
| `Esc` | 关闭弹窗/退出搜索 |

### 新增页面
| 页面 | 功能 |
|------|------|
| `about.html` | 项目介绍（中文） |
| `favorites.html` | 收藏管理 |
| `search.html` | 全文搜索 |
| `settings.html` | 个性化偏好设置 |

### 视觉优化
| 功能 | 说明 |
|------|------|
| 亮色模式修正 | 标签/徽章/链接颜色在亮色下可读 |
| 相对时间显示 | 今天/昨天/N天前/月日 |
| 打印优化 | @media print 样式，隐藏动画和装饰 |
| 导航图标 | 📅 往期回顾 / ⭐ 收藏 / 🔍 搜索 |
| 下拉刷新进度条 | 移动端下拉刷新指示器 |
