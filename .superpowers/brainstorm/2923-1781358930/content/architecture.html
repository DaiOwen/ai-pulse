⚙️ 三种实现策略，怎么选？

A

全自动化 Cron + Claude

CronCreate 定时触发 → Claude Code 自动执行全流程（搜索→处理→生成HTML）。完全无人值守，但每次运行消耗较多 token。

B

半自动 Skill 模式

写成一个 Claude Code Skill，你手动触发 "/ai-digest" 命令。灵活控制每次运行，token 消耗可控，但不自动。

C (推荐)

混合模式：Cron 调度 + 可手动补充

CronCreate 承担日常3次自动采集，同时提供手动 Skill 命令用于补充采集或重新生成。兼顾自动化和灵活性。
